Edukade Changelog

Last updated 5 August 2019

This is a changelog coded in HTML, for an app developed using MIT App Inventor 2, named Edukade.

The versioning scheme follows Semantic Versioning rules as described in semver.org 2.0.0 (retrieved 27 July 2019).

0.4.0 (3 August 2019)

  • Chapter 1 screen is now accessible
    • In the chapter select screen, pressing on ABC now redirects to the chapter 1 screen
      • The pressed-down sprite of the ABC button is yet to be added
    • In the chapter 1 screen, the return button now actually redirects to the chapter select screen, as planned
  • Removed placeholder text underneath "A" and "B" in the chapter 1 screen; now they are empty, pressable buttons that do not redirect to anywhere
  • Bug found: When pressing a button, if its pressed-down sprite does not appear but the finger is immediately lifted (making the button no longer pressed), the AI companion crashes. This has not yet been tested with the app actually compiled as an APK file.

0.3.0 (27 July 2019)

  • Added (not yet accessible) chapter 1 screen
    • Added small version of the logo (like in the chapter select screen)
    • Added books-on-a-table-in-nature background
    • Added a (not yet functioining) return button to return to the chapter select screen
    • Starting to add levels, with labels A and B placed appropriately on the bottom, and (not yet functioning) buttons underneath each one both labeled "Text" as a placeholder, later to be pressed to play the respective level

0.2.0 (23 July 2019)

  • Added chapter select screen
    • Uses the same background image as the main menu
    • Added smaller version of the logo
    • Added a return button to redirect to the main menu
    • Added a (not yet functioning) settings button
    • Added a (not yet functioning) button for chapter ABC, with a label below intended to later display the player's score in that chapter, but now containing "nilai: 100%" as placeholder text representing a score of 100%
    • Added a (not yet functioning) button for a yet-to-be-named "Chapter2", also with a score label below, currently having "nilai:" (without the 100%) as placeholder text
    • Added (not yet functioning) left arrow and right arrow buttons beside the chapter buttons, intended to later be used to navigate between chapters to select
  • The play button in the main menu screen now redirects to the chapter select screen

0.1.0 (5 July 2019)

  • Added main menu screen
    • Added background
    • Added logo
    • Added not yet functioning buttons
      • Added (not yet functioning) play button
      • Added (not yet functioning) info button
      • Added (not yet functioning) settings button
      • These buttons can already be pressed, and shows a pressed sprite accordingly, but do not redirect anywhere yet
    • Added version text, which now shows "Version: 0.1.0", and will change according to the version

(end of changelog)
